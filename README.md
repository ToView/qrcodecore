# oriboy qrcode使用文档
## 介绍
QRCodeUtils是基于Google提供的zxing二维码生成库而整合的一个生成或者解析二维码的工具类。
BarcodeUtils是基于Google提供的zxing条形码生成库而整合的一个生成或者解析条形的工具类。
## 使用方法
将qrcode-core-1.0.jar引入项目lib库，直接调用即可。
## API说明
请查看工具类javadoc注释。
